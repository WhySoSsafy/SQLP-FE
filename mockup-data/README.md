# SQLP AI Coach Mockup Data

이 폴더의 JSON 파일은 화면 확인용 mockup 데이터입니다.

- 실제 앱 저장 데이터와 섞이지 않게 하려면 사이드바의 `Mock 데이터 > 추가/삭제` 버튼을 사용하세요.
- JSON 등록 화면의 파일 업로드 흐름을 확인하려면 아래 파일 중 하나를 업로드하면 됩니다.
- 업로드한 파일은 실제 세션 저장 키에 저장됩니다. 업로드 테스트 후 지우려면 브라우저 localStorage에서 `sqlp_ai_coach.sessions.v1`를 삭제하세요.
- 사이드바 Mock 버튼으로 넣은 데이터는 `sqlp_ai_coach.mock_sessions.v1` 키만 삭제하면 사라집니다.
