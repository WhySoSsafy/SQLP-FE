SQLP 학습자를 위한 AI 기반 학습 추천·관리 웹 서비스를 만들어줘.

서비스명은 임시로 “SQLP AI Coach”로 해줘.
전체 분위기는 깔끔하고 전문적인 학습 대시보드 느낌이면 좋겠어.
색상은 화이트 배경을 기본으로 하고, 텍스트와 사이드바는 차콜 또는 다크 그레이 계열을 사용해줘.
포인트 컬러는 옐로우 계열을 사용해줘.
옐로우는 너무 형광색처럼 튀지 않게, 따뜻한 머스터드 옐로우나 골드 옐로우 느낌으로 사용해줘.
예를 들어 버튼, 강조 배지, 선택된 메뉴, 주요 카드 포인트, 캘린더 하이라이트 등에 옐로우를 적용해줘.
SQLP 자격증 학습 서비스이기 때문에 너무 귀엽기보다는 신뢰감 있고 차분한 UI로 만들어줘.

이 서비스는 사용자가 외부 AI에서 생성한 SQLP 문제 풀이 분석 JSON 결과를 등록하고, 그 데이터를 기반으로 문제별 이해도, 취약 개념, 학습 캘린더, 오답노트, 복습 추천, 스터디원 비교를 확인하는 웹 서비스야.

웹은 Django 백엔드와 Vue 프론트엔드로 구현할 예정이므로, Vue 기반 웹앱 느낌의 반응형 화면으로 설계해줘.

주요 사용자 흐름은 다음과 같아.

1. 사용자가 로그인한다.
2. 홈 대시보드에서 오늘의 복습 추천과 최근 학습 상태를 확인한다.
3. 사용자는 “JSON 분석 결과 등록” 화면으로 이동한다.
4. JSON 파일을 업로드하거나, 외부 AI가 출력한 JSON 텍스트를 직접 붙여넣는다.
5. “형식 검증하기” 버튼을 누르면 문제 수, 참여자 수, 학습 날짜, 문제집명, 주요 개념 태그가 미리보기로 나온다.
6. “학습 기록 등록하기” 버튼을 누르면 학습 세션이 저장된다.
7. 사용자는 문제별 이해도 결과, 취약 개념 대시보드, 학습 캘린더, 오답노트, 스터디원 비교 화면을 확인할 수 있다.

3. JSON 분석 결과 등록 화면

* 제목: AI 분석 결과 JSON 등록

* 설명 문구: “외부 AI에서 생성한 SQLP 풀이 분석 JSON을 업로드하거나 직접 붙여넣어 학습 기록으로 등록하세요.”

* 안내 문구를 추가해줘.

  * “문제 이미지, 정답/해설 파일, STT 텍스트, 분석 프롬프트는 외부 AI에 먼저 입력하고, 생성된 JSON 결과만 이곳에 등록합니다.”

* 등록 방식 탭 2개를 만들어줘.

  * JSON 파일 업로드
  * JSON 직접 붙여넣기

* 선택된 탭은 옐로우 밑줄 또는 옐로우 배경으로 강조해줘.

* JSON 파일 업로드 탭에는 드래그 앤 드롭 업로드 박스를 만들어줘.

  * “.json 파일을 여기에 끌어오거나 클릭해서 업로드하세요”

* 업로드 박스 테두리는 점선으로 하고, hover 또는 강조 상태는 옐로우로 표현해줘.

* JSON 직접 붙여넣기 탭에는 큰 textarea를 만들어줘.

  * placeholder로 JSON 예시 일부를 보여줘.

예시 placeholder:
{
"session_date": "2026-06-12",
"book": "SQLP 실전문제",
"speakers": ["세은", "수철"],
"problems": [...]
}

* 하단 버튼:

* “학습 기록 등록하기” 버튼은 옐로우 메인 버튼으로 만들어줘.

  * 검증 상태: 정상 / 오류

* 오류가 있을 때는 빨간색 경고 박스로 “필수 필드가 누락되었습니다” 같은 메시지를 보여줘.

* 정상일 때는 옐로우 또는 연한 그린 계열 성공 박스로 “JSON 형식이 올바릅니다. 학습 기록으로 등록할 수 있어요.”라고 보여줘.
